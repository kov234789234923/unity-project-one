# Мой Unity Проект

Проект с различными игровыми системами на Unity.

### Реализованные системы

### Управление игроком
- FPS контроллер с управлением от первого лица
- Обработка ввода с мыши и клавиатуры
- Плавное движение и вращение камеры

### Система счета
- Динамическое изменение счета
- Изменение цвета текста в зависимости от значения
- Обработка различных условий отображения

### Система камер
- Переключение между несколькими камерами
- Синхронизация с системой счета

### Портал и освещение
- Спавн порталов во время выполнения
- Динамическое создание источников света
- Изменение материалов объектов
- Система активации освещения

### Спавн объектов
- Создание объектов по клику мыши
- Управление иерархией объектов
- Очистка созданных объектов

## Технологии

- Unity 2022.3+
- C#
- Физика Unity
- Система префабов
- Работа с UI
